# 📝 To-Do List

A simple and user-friendly **To-Do List** application built using **Python**.

Manage your daily tasks directly from the terminal with a clean menu-driven interface.

---

## ✨ Features

- ➕ Add New Tasks
- 📋 View All Tasks
- 🗑️ Delete Tasks
- ✅ Input Validation
- 🔁 Menu Driven Program
- 💻 Clean Console Interface

---

## 🛠️ Technologies Used

- 🐍 Python 3
- 📋 Lists
- 🔁 While Loop
- 🔄 For Loop
- ✅ Conditional Statements

---

## 🚀 How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python todo-list.py
```
